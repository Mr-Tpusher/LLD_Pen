# Pen Version 0


## functionalities/APIs:-
1. write(String text)


### what's a Pen?
A Pen is something that has a Body, Nib and Ink.

### Types of Pens in the system:
1. BallPen
    - Has a Refill which in turn has Nib and Ink
2. InkPen
    - Has Nib and Ink
3. Marker
    - Has Nib and Ink

### Behaviours:
```write(String text)``` -> All the Pens have distinct write behaviour.
